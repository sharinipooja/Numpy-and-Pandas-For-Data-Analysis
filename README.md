# Numpy-and-Pandas-For-Data-Analysis
# Sales Data Analysis using NumPy and Pandas

## Overview

This project demonstrates basic data analysis using NumPy and Pandas on a sales dataset. Various analytical operations and visualizations are performed to understand business performance.

## Features

- CSV data loading
- Revenue analysis
- GroupBy operations
- Statistical analysis
- Data visualization
- Business insights

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Dataset

sales_data.csv

## Workflow

1. Load CSV dataset
2. Read data using Pandas
3. Perform data cleaning
4. Group sales by product category
5. Calculate total revenue
6. Generate visualizations
7. Analyze business performance

## Analysis Performed

- Total Revenue by Product Category
- Summary Statistics
- Data Visualization
